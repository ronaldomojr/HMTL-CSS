Trabalhos de HTML/CSS — Progressão Acadêmica

Descrição
--------
Este repositório reúne os trabalhos práticos realizados durante as aulas de HTML e CSS. Cada pasta dentro do projeto corresponde a uma aula ou atividade (AVD), mostrando a evolução das minhas habilidades: desde páginas estáticas básicas até estilização mais avançada, uso de fontes, imagens, vídeos e configurações de projeto.

O que foi usado
--------------
- HTML5 — marcação das páginas.
- CSS3 — estilização (arquivos `.css` presentes nas pastas das aulas).
- Editor recomendado: Visual Studio Code.
- Navegadores modernos (Chrome, Edge, Firefox) para visualização.
- Arquivos de mídia: imagens (pastas `imagens/`) e vídeos (`.ogv` em `aula 5/midia/`).
- Web App Manifest (`favicon/site.webmanifest`) para metadados do site.
- Fontes e licenças (pasta `Fontes/` com `OFL.txt`).

Propósito
---------
Esses arquivos são trabalhos e exercícios feitos ao longo das aulas da faculdade. Servem para demonstrar a minha progressão prática em desenvolvimento front-end: aprender marcação semântica, layout, responsividade básica, organização de recursos (imagens, fontes e manifest) e boas práticas simples.

Estrutura do repositório
------------------------
- `apw1/` — pastas organizadas por aula (aula 2, aula 3, ..., aula 9). Cada uma contém um `index.html` e recursos associados.
- `AVD-HTML-CSS/` — atividade avaliativa com arquivos base e estilos.

Exemplos rápidos de caminhos
----------------------------
- Abra `apw1/aula 2/index.html` para ver o trabalho da aula 2.
- Abra `apw1/aula 5/index.html` para ver páginas com mídias (vídeo e imagens).
- Verifique `apw1/aula 3/favicon/site.webmanifest` para o manifest do site.

Como visualizar
---------------
1. No Explorador de arquivos, abra o arquivo `index.html` da aula desejada no navegador.
2. Recomendo usar a extensão Live Server do VS Code para desenvolver e visualizar com reload automático.

Notas finais
-----------
Este repositório é um registro do meu aprendizado prático. Se desejar, posso:
- Gerar um sumário por aula.
- Organizar os trabalhos em uma apresentação ou site hospedado.
